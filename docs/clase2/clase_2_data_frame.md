## Data Frame

* Un **`data.frame` es un **tabla**.
* Es una **matriz indexada**: las columnas y filas tienen **nombre**.
* Cada **columna** es una **variable**.
* Cada **fila** es una **observación**.
* También es un conjunto de **vectores**.
